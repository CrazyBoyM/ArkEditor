# ArkEditor
方舟编辑器（Typecho写作插件）  

![预览](https://es.ip3x.com/ArkEditor/ArkEditor01.jpg)
![预览2](https://es.ip3x.com/ArkEditor/ArkEditor02.jpg)

## 功能清单

- 自动补全  
- 智能提示  
- 实时预览
- 缓存文章
- 图片预览
- 脑图
- 流程图
- 时序图
- 数学公式
- 。。。

## 计划开发清单

- 文本智能纠错
- 优化预览布局
- 改进界面风格
- 图片上传
- 表情插入
- 文章分享到方舟、南村

## API

- - 百度NLP算法接口 (用于实现插件中的一些智能辅助)

## 基于以下项目    

- ace.js  
- marked.js  
- layui.js  
- Jquery.js
